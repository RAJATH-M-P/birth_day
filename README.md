# Birthday Wishes Page

This repository contains a simple birthday celebration web page.

## Files

- `birthday.html` — the birthday greeting page with animations and a Celebrate button.
- `birthday-song.mp3` — the audio file used by the page when the celebration starts.

## How to use

1. Open `birthday.html` in a browser.
2. Click the **Celebrate!** button.
3. The page will show confetti, balloons, and play the birthday song.

## GitHub Pages

To share this page as a URL, publish the repository using GitHub Pages.

If the repo is owned by `RAJATH-M-P` and named `birth_day`, the URL will be:

`https://rajath-m-p.github.io/birth_day/birthday.html`

Note: The browser may require the visitor to click the Celebrate button before audio playback starts.
